#  Einschränkungen


**Einstellungen -> CommonsBooking -> Reiter: Einschränkungen**

Hier verwaltest Du die Vorlagen zu den E-Mails, die bei Einschränkungen an die Ausleihenden versendet werden, die eine Buchungs im Einschränkungszeitraum haben.

Du kannst TemplateTags (Platzhalter) verwenden. Siehe in der [ Liste der
Template-Tags ](../administration/template-tags) nach, um passende
Platzhalter zu finden.

Wie du [Einschränkungen verwendest ist hier beschrieben.](../first-steps/manage-booking-restrictions)

Darüber hinaus kannst du hier die Buchungsregeln konfigurieren. Wie du [Buchungsregeln verwendest ist hier beschrieben.](../first-steps/setup-bookingrules)
