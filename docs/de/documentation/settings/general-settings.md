#  Allgemeine Einstellungen

__

Unter Einstellungen -> CommonsBooking findest du den Reiter **Allgemein**

  * **Slug**
    * Passe hier die Slugs für deine Buchungs-URLs an. Das kann nützlich sein, wenn die URLs in einem bestimmten Format angezeigt werden sollen (z.B. `https://example.com/artikel/artikelname`)
  * **Buchungsseite**
    * Wähle hier die Seite aus, die im [ User Widget ](/documentation/administration/widget) als Buchungsliste angezeigt werden soll, wenn die Nutzenden auf "Meine Buchungen" klicken. Das ist im Normalfall eine Seite, auf der der cb_bookings Shortcode eingebunden ist. 
  * **Buchungskommentar**
    * Stelle ein, ob Ausleihende bei einer Buchung einen Kommentar hinterlassen können. Dieser Kommentar wird an die Buchung angehängt.
  * **Globale Standorteinstellungen**
    * Verändere das Standardverhalten für die Überbuchung bestimmter Schließtage für alle Standorte. Diese Einstellung definiert nur das Standardverhalten und kann von jedem Standort individuell überschrieben werden. Mehr Informationen dazu findest du unter [Standorteinstellungen: Überbuchung]( /documentation/first-steps/create-locations#overbooking ) .
